Cómo empezar en CSS

1. ¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje que usamos para dar estilo y diseño a las páginas web.

body {
  background: #f4f6f8;
  color: #333;
  font-family: Arial, sans-serif;
}

2. Cómo vincular CSS a tu HTML

Usa la etiqueta <link> en el <head>:

<link rel="stylesheet" href="estilos.css">

3. Selectores básicos

Selecciona elementos por etiqueta, clase o id:

h1 {
  color: #2563eb;
}
.caja {
  border: 1px solid #ccc;
}
#principal {
  padding: 2rem;
}

4. Box Model

El modelo de caja determina cómo se calculan el tamaño y el espacio de los elementos (content, padding, border, margin).

/* Box model */
.caja {
  width: 200px;
  padding: 16px;
  border: 2px solid #ccc;
  margin: 12px;
  box-sizing: border-box; /* incluye padding y border en el ancho */
}

5. Posicionamiento

Propiedades: static, relative, absolute, fixed y sticky.

/* Posicionamiento */
.box-abs {
  position: absolute;
  top: 20px;
  left: 20px;
}
.box-rel { position: relative; }

6. Flexbox

Flexbox es útil para distribuir espacio en una dimensión (fila o columna).

/* Flexbox */
.contenedor {
  display: flex;
  gap: 12px;
  align-items: center;
}
.item { flex: 1; }

7. Grid

CSS Grid permite crear layouts en dos dimensiones de forma sencilla.

/* Grid */
.grid {
  display: grid;
  grid-template-columns: 1fr 2fr;
  gap: 16px;
}
.grid > div { padding: 8px; background: #f3f4f6; }

8. Responsive y Media Queries

Usa media queries para adaptar el diseño según el tamaño de la pantalla.

/* Responsive */
@media (max-width: 600px) {
  .grid { grid-template-columns: 1fr; }
  .contenedor { flex-direction: column; }
}

9. Variables CSS

Las custom properties facilitan mantener valores reutilizables.

/* Variables */
:root {
  --primary: #2563eb;
  --gap: 12px;
}

Unidades relativas (%, em, rem, vw, vh) y absolutas (px).

11. Transiciones y Animaciones

Transiciones para cambios suaves y @keyframes para animaciones complejas.

/* Transición */

12. Especificidad

La especificidad decide qué reglas aplican cuando hay conflicto entre selectores.

/* Especificidad */

13. Pseudo-clases y Pseudo-elementos

Permiten seleccionar estados y partes de elementos (hover, before, after).

/* Pseudo */
.btn { background: var(--primary); padding: var(--gap); color: #fff; }

14. Combinadores

Seleccionan relaciones entre elementos (descendiente, hijo, adyacente).

/* Combinadores */

15. Tipografía

Control de fuentes, interlineado y kerning.

/* Tipografía */

16. Colores

Formas de definir colores: hex, rgb(a), hsl(a).

/* Colores */
.texto { font-size: 1rem; /* 1rem = tamaño de la fuente raíz */ }
.caja { width: 50%; }

17. Gradientes

Gradientes lineales y radiales para fondos más ricos.

/* Gradientes */
background: linear-gradient(to right, #2563eb, #1d4ed8);

18. Fondos

Control de imágenes, repetición y posición de fondo.

/* Fondos */
.fondo {
  background-image: url('fondo.jpg');
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
}

19. Sombras

Box-shadow y text-shadow para profundidad.

/* Sombras */
.caja {
  box-shadow: 0 4px 6px rgba(0,0,0,0.1);
}
.titulo {
  text-shadow: 1px 1px 2px #aaa;
}

20. Bordes

Control del estilo, radio y grosores de bordes.

/* Bordes */
.boton {
  border-radius: 8px;
  border: 2px solid var(--primary);
  transition: background 200ms ease;
}
.boton:hover {
  background: #1d4ed8;
  color: #fff;
}

21. Columnas (multi-column layout)

Divide texto en columnas con CSS columns.

/* Columnas */
.multicolumn {
  column-count: 3;
  column-gap: 1rem;
}

22. Filtros (filter)

Efectos visuales como blur, saturate o grayscale.

/* Filtros */
.filter-blur {
  filter: blur(4px);
}
.filter-grayscale {
  filter: grayscale(100%);
}

23. Transformaciones (transform)

Translate, rotate, scale y transform-origin para mover elementos.

/* Transform */
.transformar {
  transform: rotate(15deg) translateX(20px);
  transform-origin: center center;
}

24. Cálculos con calc()

Combina unidades y cálculos en propiedades CSS.

/* calc */
.calculado {
  width: calc(100% - 40px);
}

25. clamp() y funciones útiles

Controla valores responsivos con clamp(min, val, max).

/* clamp */
.texto-responsive {
  font-size: clamp(1rem, 2vw, 1.5rem);
}

26. object-fit / object-position

Cómo ajustar imágenes dentro de contenedores.

/* object-fit */
.imagen-ajustada {
  object-fit: cover;
  object-position: center;
}

27. Imágenes responsivas (srcset/CSS)

Buenas prácticas para servir imágenes adecuadas al tamaño.

/* responsive images (CSS example) */
img {
  max-width: 100%;
  height: auto;
}

28. pointer-events y visibility

Control de interacción y visibilidad sin afectar layout.

/* interacción */
.elemento {
  pointer-events: none;
  visibility: hidden;
}

29. Estilado de formularios

Mejorar inputs, selects y botones para una UI consistente.

/* Formularios */
input, select, button {
  padding: 0.5rem 1rem;
  border-radius: 6px;
  border: 1px solid #ccc;
}

30. CSS para impresión

Usa media=print para adaptar el contenido a papel.

/* Print */
@media print {
  body { font-size: 12pt; color: #000; background: #fff; }
  .sidebar { display: none; }
}

31. Accesibilidad en CSS

Contraste, focus-visible y tamaños legibles ayudan a la accesibilidad.

/* Accesibilidad */
:focus-visible {
  outline: 2px solid var(--primary);
  outline-offset: 2px;
}

32. Rendimiento y buenas prácticas

Minimiza selectores costosos, usa archivos optimizados y cache.

/* Prácticas */
body {
  will-change: transform;
  contain: layout style;
}

33. Container Queries

Permiten aplicar estilos basados en el tamaño del contenedor padre, no solo de la ventana.

/* Container Queries */
.contenedor {
  container-type: inline-size;
}
@media (min-width: 400px) {
  .contenedor .elemento { font-size: 1.2rem; }
}

34. Propiedades Lógicas de CSS

Usa propiedades lógicas como margin-inline-start en lugar de margin-left para soporte RTL.

/* Propiedades lógicas */
.elemento {
  margin-inline-start: 1rem;
  padding-block-end: 0.5rem;
}

35. CSS Nesting

Permite anidar selectores dentro de otros para un código más organizado.

/* Nesting */
.padre {
  color: blue;
  .hijo {
    color: red;
  }
}

36. CSS Scoping

Limita el alcance de los estilos con @scope para evitar conflictos globales.

/* Scoping */
@scope (.componente) {
  .titulo { font-size: 1.5rem; }
}

37. CSS Houdini

API para extender CSS con propiedades personalizadas y efectos avanzados.

/* Houdini (ejemplo básico) */
CSS.registerProperty({
  name: '--mi-propiedad',
  syntax: '',
  initialValue: '0px'
});

38. CSS Containment

Optimiza el rendimiento aislando el layout de subárboles.

/* Containment */
.contenedor {
  contain: layout style paint;
}

39. CSS Scroll Snap

Crea experiencias de scroll suaves y precisas en galerías o secciones.

/* Scroll Snap */
.galeria {
  scroll-snap-type: x mandatory;
}
.galeria > div {
  scroll-snap-align: start;
}

40. CSS Writing Modes

Cambia la dirección del texto y layout (horizontal, vertical).

/* Writing Modes */
.vertical {
  writing-mode: vertical-rl;
}

41. CSS Shapes

Permite que el texto fluya alrededor de formas no rectangulares.

/* Shapes */
.imagen {
  shape-outside: circle(50%);
}

42. CSS Blend Modes

Mezcla colores de elementos superpuestos con modos como multiply o screen.

/* Blend Modes */
.superpuesto {
  mix-blend-mode: multiply;
}

43. CSS Masks

Oculta partes de elementos usando imágenes o gradientes.

/* Masks */
.elemento {
  mask-image: linear-gradient(to right, transparent, black);
}

44. CSS Custom Scrollbars

Personaliza el aspecto de las barras de scroll.

/* Custom Scrollbars */
.scrollable::-webkit-scrollbar {
  width: 8px;
}
.scrollable::-webkit-scrollbar-thumb {
  background: #2563eb;
}

45. CSS Aspect Ratio

Mantiene proporciones de elementos sin especificar altura.

/* Aspect Ratio */
.video {
  aspect-ratio: 16 / 9;
}

46. Funciones Matemáticas de CSS

Usa funciones como min(), max(), round() para cálculos avanzados.

/* Math Functions */
.elemento {
  width: min(100%, 500px);
  font-size: max(1rem, 2vw);
}

47. Funciones de Color de CSS

Manipula colores con funciones como color-mix() o color-contrast().

/* Color Functions */
.elemento {
  color: color-mix(in srgb, red 50%, blue 50%);
}

48. CSS Grid Subgrid

Permite que grids hijos hereden la cuadrícula del padre.

/* Subgrid */
.padre {
  display: grid;
  grid-template-columns: 1fr 2fr;
}
.hijo {
  grid-column: 1 / -1;
  display: grid;
  grid-template-columns: subgrid;
}

49. CSS Modules

Enfoque modular para estilos, evitando conflictos de nombres.

/* CSS Modules (concepto) */
/* En un archivo .module.css */
.titulo { font-size: 2rem; }
/* Se importa como objeto en JS */
import styles from './styles.module.css';

50. CSS-in-JS

Escribe estilos en JavaScript para estilos dinámicos y scoped.

/* CSS-in-JS (ejemplo con styled-components) */
const Boton = styled.button`
  background: ${props => props.primary ? 'blue' : 'gray'};
  color: white;
`;

51. CSS Animation Libraries

Librerías como Animate.css para animaciones predefinidas.

/* Animate.css */
.elemento {
  animation: bounce 2s infinite;
}

52. CSS Custom Properties Avanzadas

Usa variables CSS con fallback y cálculos dinámicos.

/* Custom Properties Avanzadas */
:root {
  --espacio: 1rem;
}
.elemento {
  padding: var(--espacio, 0.5rem);
  margin: calc(var(--espacio) * 2);
}
