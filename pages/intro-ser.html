¿Qué es una Red Informática?

Comunicación

Una red informática conecta ordenadores y dispositivos para compartir información y recursos.

Compartición

Los usuarios pueden compartir archivos, impresoras, aplicaciones y acceso a Internet.

Ventajas

Reduce costes, mejora la administración y aumenta la seguridad de los datos.

Configuración TCP/IP

Para que los dispositivos se comuniquen en una red, necesitan una dirección IP y una máscara de subred.

Dirección IP

Identifica de forma única a cada equipo dentro de la red.

Máscara de Subred

Define el tamaño de la red y qué equipos pertenecen a ella.

Puerta de Enlace

Permite que los dispositivos puedan acceder a Internet.

DNS

Traduce nombres de dominio en direcciones IP.

Ejemplo de configuración:

IP: 192.168.0.10

Máscara: 255.255.255.0

Gateway: 192.168.0.1

DNS: 8.8.8.8

Direcciones IP Públicas y Privadas

IP Privada

Se utiliza dentro de redes locales y no es accesible directamente desde Internet.

IP Pública

Es la dirección visible en Internet y normalmente pertenece al router.

Rangos Privados

192.168.X.X
172.16.X.X
10.X.X.X

NAT - Traducción de Direcciones

NAT permite que múltiples dispositivos de una red local compartan una única dirección IP pública.

SNAT

Cambia la IP privada por una IP pública para acceder a Internet.

Seguridad

Oculta las IPs internas de la red local.

Ahorro de IPs

Permite que muchos equipos usen una sola IP pública.

Servicios de Red

DHCP / DNS

Automatizan la configuración de red y resolución de nombres.

SSH

Permite acceso remoto seguro a servidores.

FTP

Facilita la transferencia de archivos en la red.

WEB

Proporciona páginas y aplicaciones accesibles desde navegadores.

EMAIL

Servicio de comunicación mediante correo electrónico.

Impresión

Permite compartir impresoras entre varios equipos.

Servidores y Hardware

Servidor

Equipo encargado de ofrecer servicios y recursos a otros dispositivos de la red.

RAM y CPU

Los servidores requieren mucha memoria RAM y procesadores potentes.

Almacenamiento

Utilizan discos de alta capacidad y velocidad.

Refrigeración

Necesitan sistemas avanzados de ventilación para funcionar 24/7.
